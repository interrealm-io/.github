# InterRealm

**Open source coordination protocol for distributed AI agents**

InterRealm enables autonomous AI agents to collaborate across networks with cryptographic verification. Apache 2.0 licensed.

---

## What It Does

InterRealm provides infrastructure for distributed AI coordination:

- **Decentralized agent discovery** through capability schemas
- **Recursive coordination** where agents spawn sub-agents dynamically  
- **Policy-based governance** via hierarchical realm structures
- **Cryptographic verification** with blockchain identity for every agent
- **Real-time monitoring** through the Nexus Console
